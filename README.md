# Sentiment Analysis with LSTM

This project demonstrates sentiment analysis using an LSTM network, incorporating TF-IDF and word embeddings for text classification.

## Requirements

- Python 3.7+
- TensorFlow 2.0+
- Numpy
- Pandas
- Scikit-learn

- Gensim
- Matplotlib
